An Introduction to IPython Notebook

"The IPython Notebook is a web-based interactive computational environment where you can combine code execution, text, mathematics, plots and rich media into a single document"

ipython.org/notebook

IPython notebooks are google docs that can execute python¶

Notebooks are made up of independent "cells"

  • Markdown
  • Latex
  • Python code, and its output

When code is organized in cells it can be executed sequentially or individually¶

In [2]: 
def small_part_of_problem(x, y):
    return x + y

Plotting User Registration

In [13]: 
from django.db.models import Count
from django.contrib.auth.models import User

# Get the day off of the datetime field as an integer
queryset = User.objects.extra(select={'day': 'date( date_joined )'})

# Values does the equivalent of a GROUP BY
queryset = queryset.values('day')

# Annotate with the count of Users who joined on that day
queryset = queryset.annotate(joined_count=Count('date_joined'))

# Order By the day
queryset = queryset.order_by('day')
In [28]:

How I've used IPython Notebooks for projects¶

Exploratory

  • Notebooks can be saved easily so when I'm doing exploratory work I crack open a notebook instead of a shell or text editor

IPython Notebooks are postits for Python and Django Developers¶

Semi-frequent Questions

  • Notebooks can hold onto logic that doesn't need to be in the app but that creates business value

Data Interfaces for Very Trusted Users

  • I can give a notebook to a member of the marketing team with some examples and they can answer their own questions

Tripp

  • Interested in python but not a developer
  • Needs questions answered about marketing traction

A setup I've used for remote access

  • IPYTHON_ARGUMENTS += ['--ip="*"']
  • run python manage.py shell_plus --notebook under supervisord
  • Run nginx in front of this with LDAP/PAM authorization
  • Have deployment process push new code to the directory where notebook is running and restart the process so you can access new models

Caveats

  • Don't do analysis on production data, use a read slave instead
  • This has the full power of python, don't give to anyone you don't trust
